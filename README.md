# Not Escape Room 🔓

脱出不可能？いや、脱出**不要**かもしれません。

## ゲーム説明

このゲームは、一見すると脱出するのに1年かかる悪質な脱出ゲームです。しかし、賢い人は別の方法を見つけるでしょう...

### 通常プレイ
- 複雑なパズルを解く
- アイテムを集める
- 暗号を破く

**推定クリア時間：約20分**

### 謎解きシステム
このゲームには **5つの異なるパズル** があります：

1. **暗号解読パズル** (入口)
   - ROT13暗号を解く必要があります
   - ヒント: 13文字シフト

2. **スフィンクスの謎** (パズルの部屋)
   - 古典的な謎: 朝は4本足、昼間は2本足...
   - ⭐ **このパズルを解かないと先に進めません**

3. **計算問題** (物置)
   - 数学的なパズル

4. **シーケンス判定** (図書館)
   - シンボルのパターンを見つける
   - ⭐ **脱出口に進むには必須**

5. **コード解読** (脱出口)
   - アルファベット→数字対応を使う

### 裏技の存在
このゲームには **隠された脱出方法** があります。
- 環境変数
- 特殊なコマンド
- イースターエッグ
- ...etc

探してみてください！🔍

## 遊び方

### キーボード操作
- **W** - 北へ移動
- **A** - 西へ移動
- **S** - 南へ移動
- **D** - 東へ移動
- **T** - アイテムを取得
- **I** - インベントリ確認
- **H** - ヒント表示
- **R** - ゲームをリセット
- **Enter** - パズルの答えを送信

### マウス操作
- 方向キーボタンをクリック
- 「取得」「インベントリ」などのボタンをクリック
- パズルの答えを入力して「回答」ボタンをクリック

## ゲーム構成

```
入口 (暗号解読パズル)
├── 北 → パズルの部屋 (スフィンクスの謎) ← 【必須】
│       └── 東 → 図書館 (シーケンス判定)
│               └── 東 → 脱出口 (コード解読) ← 【最終パズル】
├── 東 → 物置 (計算問題)
└── アイテム: 鍵、メモ
```
