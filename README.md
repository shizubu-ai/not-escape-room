# Not Escape Room 🔓

脱出不可能？いや、脱出**不要**かもしれません。

## ゲーム説明

このゲームは、一見すると脱出するのに1年かかる悪質な脱出ゲームです。
ただし、観察力のある人にはショートカットが見えるかもしれません。

## 特徴

### 通常プレイ（超高難度）
- 複数ルームを移動しながら解除する構成
- 12桁・10桁のロック解除
- 大量順列を想定したパターンパズル
- ルーム観察・アイテム使用・入力推理が必要

### 隠しショートカット（裏技）
- 隠しコマンド
- デバッグ系の挙動
- 環境変数の利用余地
- テキストに紛れたイースターエッグ
- 特殊キーやシーケンスの抜け道

## 実行方法

### ブラウザで遊ぶ（推奨）

**GitHub Pages で公開中:**
👉 [Not Escape Room をプレイ](https://shizubu-ai.github.io/not-escape-room/)

または、`index.html` をブラウザで開いてください：
```bash
# リポジトリをクローン
git clone https://github.com/shizubu-ai/not-escape-room.git
cd not-escape-room

# ブラウザで開く
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

## 基本コマンド

- `look` - 周囲を観察
- `go [direction]` - 移動
- `take [item]` - アイテムを取得
- `use [item] [target]` - アイテムを使用
- `solve [puzzle] [answer]` - パズル回答
- `inventory` - インベントリ確認
- `hint` - ヒント表示
- `help` - コマンド一覧
- `quit` - 終了

## ヒント（ネタバレなし）

- 部屋説明の不自然な文言を見逃さない
- 伝統的なゲーム文化にある“お約束”を試してみる
- 環境変数を使うゲームは昔から存在する
