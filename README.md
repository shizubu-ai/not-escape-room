# Not Escape Room 🔓

脱出不可能？いや、脱出**不要**かもしれません。

## ゲーム説明

このゲームは、一見すると脱出するのに1年かかる悪質な脱出ゲームです。しかし、賢い人は別の方法を見つけるでしょう...

### 通常プレイ
- 複雑なパズルを解く
- アイテムを集める
- 暗号を破く

**推定クリア時間：約1年**

### 裏技の存在
このゲームには **隠された脱出方法** があります。
- 環境変数
- 特殊なコマンド
- イースターエッグ
- ...etc

探してみてください！🔍

## 実行方法

```bash
python main.py
```

## 遊び方

1. ゲームを起動する
2. コマンドを入力して行動する
3. パズルを解いて脱出を目指す
4. または、別の方法を見つけてみる...

## コマンド例

- `look` - 周囲を観察
- `take [item]` - アイテムを取得
- `use [item] [target]` - アイテムを使用
- `inventory` - インベントリを確認
- `go [direction]` - 移動
- `hint` - ヒントを表示
